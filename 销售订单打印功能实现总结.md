### 4. 新增方法

#### handlePrint() - 打开打印预览
```javascript
async handlePrint(row) {
  try {
    // 获取订单完整详情
    const res = await getOrderDetail(row.orderNo)
    if (!res || res.code !== 200) {
      this.$message.error('获取订单详情失败')
      return
    }
    this.currentPrint = res.data
    this.printVisible = true
  } catch (e) {
    console.error('打开打印预览异常:', e)
    this.$message.error('打开打印预览失败')
  }
}
```

#### formatSpec() - 格式化规格显示
```javascript
formatSpec(item) {
  const thickness = item.thickness ? Math.round(item.thickness * 1000) : 0 // mm转μm
  const width = item.width ? Math.round(item.width) : 0 // mm
  const length = item.length ? Math.round(item.length / 1000) : 0 // mm转m
  return `${thickness}μm×${width}mm×${length}m`
}
```

#### formatNumber() - 格式化数字显示
```javascript
formatNumber(value) {
  if (value == null || value === '') return '0.00'
  return Number(value).toFixed(2)
}
```

#### handlePrintBrowser() - 浏览器打印
```javascript
handlePrintBrowser() {
  this.$nextTick(() => {
    window.print()
  })
}
```

### 5. 新增计算属性

#### sumRolls - 计算卷数合计
```javascript
sumRolls() {
  if (!this.currentPrint || !this.currentPrint.items) return 0
  return this.currentPrint.items.reduce((sum, item) => sum + (Number(item.rolls) || 0), 0)
}
```

#### sumArea - 计算面积合计
```javascript
sumArea() {
  if (!this.currentPrint || !this.currentPrint.items) return '0.00'
  const sum = this.currentPrint.items.reduce((s, item) => s + (item.sqm || this.calcSqmNumber(item)), 0)
  return sum.toFixed(2)
}
```

#### sumAmount - 计算金额合计
```javascript
sumAmount() {
  if (!this.currentPrint || !this.currentPrint.items) return '0.00'
  const sum = this.currentPrint.items.reduce((s, item) => {
    const amount = item.amount || (this.calcSqmNumber(item) * (Number(item.unitPrice) || 0))
    return s + amount
  }, 0)
  return sum.toFixed(2)
}
```

## 🎯 规格标准化说明

### 数据库存储（不变）
- `thickness`: DECIMAL类型，单位mm（例如：0.095）
- `width`: DECIMAL类型，单位mm（例如：1280）
- `length`: DECIMAL类型，单位mm（例如：100000）

### 前端显示（新规范）
- **厚度显示**: `Math.round(thickness * 1000)` → 95μm
- **宽度显示**: `Math.round(width)` → 1280mm
- **长度显示**: `Math.round(length / 1000)` → 100m
- **规格格式**: `95μm×1280mm×100m`

### 面积计算（保持不变）
```javascript
sqm = (length_mm * width_mm * rolls) / 1000000
```

## 📝 使用说明

### 操作步骤
1. 进入销售订单管理页面
2. 在订单列表中找到需要打印的订单
3. 点击该订单操作列的"打印"按钮
4. 系统弹出打印预览对话框，显示完整的销售合同
5. 检查合同内容是否正确
6. 点击"打印/导出PDF"按钮
7. 在浏览器打印对话框中：
   - 选择打印机或保存为PDF
   - 调整打印设置（纸张、方向等）
   - 点击打印或保存

### 注意事项
1. **规格显示**：现在规格统一显示为 厚度μm×宽度mm×长度m 格式
2. **数据库存储**：虽然前端显示有单位转换，但数据库存储仍保持原有格式（都是mm）
3. **面积计算**：自动根据长、宽、卷数计算生产方数
4. **打印设置**：建议使用A4纸张，纵向打印
5. **导出PDF**：在打印对话框中选择"保存为PDF"即可导出PDF文件
