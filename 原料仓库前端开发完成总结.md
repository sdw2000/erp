- `getAvailableChemicalDetails(id)` - 查询可用明细
- `getExpiringChemicals(days)` - 查询即将过期
- `getChemicalOutboundBySchedule(scheduleId)` - 查询出库记录

## 四、集成到排程功能

### 下一步工作

1. **修改排程对话框**
   - 在排程创建对话框中引入FilmSelector组件
   - 用户创建涂布排程时选择薄膜
   - 将选择的薄膜信息传递给后端

2. **后端集成**
   - 修改`ProductionScheduleServiceImpl.generateCoatingTasks()`
   - 接收前端传递的薄膜库存ID
   - 调用`FilmStockService.lockStock()`锁定薄膜
   - 将薄膜信息保存到`schedule_coating`表

3. **出库记录**
   - 涂布任务完成时，调用`FilmStockService.outbound()`
   - 创建薄膜出库记录
   - 更新明细状态为"已使用"

### 排程对话框示例代码

```vue
<template>
  <el-dialog title="创建排程" :visible.sync="visible" width="70%">
    <el-form :model="scheduleForm" label-width="120px">
      <!-- 原有排程信息 -->
      <el-form-item label="排程日期">
        <el-date-picker v-model="scheduleForm.scheduleDate" type="date" />
      </el-form-item>
      
      <!-- 新增：选择薄膜 -->
      <el-form-item label="选择基材薄膜">
        <film-selector
          :default-thickness="orderItem.baseThickness"
          :default-width="orderItem.baseWidth"
          @select="handleFilmSelect"
          ref="filmSelector"
        />
      </el-form-item>
      
      <!-- 其他表单项 -->
    </el-form>
    
    <div slot="footer">
      <el-button @click="visible = false">取消</el-button>
      <el-button type="primary" @click="handleSubmit">确定</el-button>
    </div>
  </el-dialog>
</template>

<script>
import FilmSelector from '@/components/FilmSelector'

export default {
  components: { FilmSelector },
  data() {
    return {
      visible: false,
      scheduleForm: {
        scheduleDate: null,
        filmStockId: null // 薄膜库存ID
      }
    }
  },
  methods: {
    handleFilmSelect(film) {
      this.scheduleForm.filmStockId = film.id
      console.log('已选择薄膜:', film)
    },
    
    async handleSubmit() {
      // 提交排程，包含filmStockId
      const res = await autoSchedule({
        ...this.scheduleForm,
        filmStockId: this.scheduleForm.filmStockId
      })
      // ...
    }
  }
}
</script>
```

## 七、使用说明

### 1. 薄膜仓库

1. 打开"薄膜仓库"页面
2. 可以查看所有薄膜库存的汇总统计
3. 在搜索框输入厚度(μm)和宽度(mm)进行筛选
4. 点击"查看明细"可以查看该物料的所有批次明细
5. 明细中显示批次号、卷号、面积、质检状态、库位等信息

### 2. 化工仓库

1. 打开"化工仓库"页面
2. 可以查看所有化工库存的汇总统计
3. 在搜索框选择化工类型进行筛选
4. 点击"查看明细"可以查看该物料的所有批次明细
5. 明细中显示批次号、桶号、重量、有效期、危险等级等信息
6. 点击"查看即将过期"按钮查看30天内即将过期的化工原料

### 3. 涂布排程选择薄膜（待集成）

1. 在排程对话框中显示薄膜选择组件
2. 输入厚度和宽度查询可用薄膜
3. 点击某一行选择该薄膜
4. 提交排程时将薄膜ID传递给后端
