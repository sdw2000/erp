# 原材料导入模板填写一页纸

## 1. 适用范围
用于 ERP【研发-原材料表】的 Excel 导入模板填写，支持新增/更新原材料。

## 2. 导入规则（最重要）
- 以 **物料代码/物料编码** 作为唯一键：
  - 已存在：更新
  - 不存在：新增
- 必填：**物料代码（或物料编码）**、**物料名称**
- 状态支持：`启用` / `禁用` / `1` / `0`（空值默认启用）
- 兼容旧列：如果“十一项性能参数列”都为空，会回退使用旧列“性能参数(JSON/范围)”

## 3. 模板字段顺序
1) 供应商代码  
2) 物料代码  
3) 物料大类  
4) 物料类别(原始)  
5) 物料类型  
6) 物料名称  
7) 单位  
8) 规格说明  
9) 备注  
10) 厚度  
11) 电晕值  
12) 抗拉强度  
13) 伸长率  
14) 颜色  
15) 外观  
16) 固含量  
17) 粘度  
18) 剥离强度  
19) 离型力A  
20) 离型力B  
21) 状态

## 4. 十一项性能参数怎么填
可填 3 种格式：
- 固定值：如 `红色`、`23μm`
- 区间：如 `120~150N/15mm`、`20~35gf/in`
- 阈值：如 `≥48dyne`、`≤2200cps`

建议单位：
- 厚度：`μm`
- 电晕值：`dyne`
- 抗拉强度：`N/15mm`
- 伸长率：`%`
- 固含量：`%`
- 粘度：`cps`
- 剥离强度：`N/25mm`
- 离型力A/B：`gf/in`

## 5. 常见问题
### Q1：我只维护了离型力A/B，其他性能列空着可以吗？
可以。系统会仅写入你填写的性能项。

### Q2：颜色、外观是文本，能导入吗？
可以，按文本值比较与展示。

### Q3：为什么导入后页面显示为 `-`？
通常是该项未填写、或单元格有隐藏空格/特殊字符。建议清理后重导。

## 6. 最小示例
- 物料代码：`PETM-T23`
- 物料名称：`23μm PET膜`
- 单位：`m²`
- 厚度：`21~25μm`
- 电晕值：`≥48dyne`
- 离型力A：`20~35gf/in`
- 离型力B：`30~45gf/in`
- 状态：`启用`
